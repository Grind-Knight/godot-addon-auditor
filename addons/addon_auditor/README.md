# Godot Add-on Auditor

Godot Add-on Auditor is a free Godot 4 editor add-on that scans editor plugins for common release and Asset Library packaging issues before you publish them.

Enable the plugin from **Project > Project Settings > Plugins**, then use the **Add-on Auditor** dock to scan the current project. It checks `plugin.cfg` metadata, plugin script paths, editor plugin script basics, missing add-on README/license files, and noisy folders that should not ship in release archives.

The project also includes a command-line version for CI and release scripts.
